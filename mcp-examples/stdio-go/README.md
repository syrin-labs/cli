# Stdio MCP Server (Go)

Go implementation of a stdio MCP server.

## Setup

```bash
go mod download
```

## Build

```bash
go build -o server
```

## Run

```bash
./server
```

## Tools

- `factorial` - Calculate factorial of a number
- `fibonacci` - Calculate nth Fibonacci number

## Prompts

- `optimize` - Optimize code

## Resources

- `example://welcome` - Welcome message
