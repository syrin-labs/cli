# HTTP/SSE MCP Server (TypeScript)

TypeScript implementation of an HTTP/SSE MCP server.

## Setup

```bash
npm install
npm run build
```

## Run

```bash
npm start
# or
npm run dev
```

Server runs on `http://localhost:8000/mcp`

## Tools

- `add` - Add two numbers
- `multiply` - Multiply two numbers

## Prompts

- `code_review` - Generate a code review prompt

## Resources

- `example://info` - Server information
