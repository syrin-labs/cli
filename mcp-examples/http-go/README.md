# HTTP/SSE MCP Server (Go)

Go implementation of an HTTP/SSE MCP server using Gin.

## Setup

```bash
go mod download
```

## Build

```bash
go build -o server
```

## Run

```bash
./server
```

Server runs on `http://localhost:8002/mcp`

## Tools

- `reverse` - Reverse a string
- `length` - Get length of a string

## Prompts

- `refactor` - Refactor code

## Resources

- `example://info` - Server information
