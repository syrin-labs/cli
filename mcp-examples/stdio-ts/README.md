# Stdio MCP Server (TypeScript)

TypeScript implementation of a stdio MCP server.

## Setup

```bash
npm install
npm run build
```

## Run

```bash
npm start
# or
npm run dev
```

## Tools

- `subtract` - Subtract two numbers
- `divide` - Divide two numbers

## Prompts

- `documentation` - Generate documentation for code

## Resources

- `example://help` - Help information
